To customize:
- Open the notify.js file in a text editor
- Customize the fields at the top of the file. DO NOT EDIT ANY OTHER PART OF THE FILE.
- Save when done

To run:
- Open a terminal in this directory
- Execute the script: node autoPageRefresh.js
- Leave the window open (can be in the background)
- Computer must be running in order for this program to continue processing

To stop:
- Open a terminal in this directory
- Stop the script: ctrl + c
